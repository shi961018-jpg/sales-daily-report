# 奶茶店销售日报生成器操作说明

## 打开方式

公网地址：

https://shi961018-jpg.github.io/sales-daily-report/

演示地址：

https://shi961018-jpg.github.io/sales-daily-report/?demo=1

## 每天怎么用

1. 打开网页。
2. 选择当天日期。
3. 输入每个饮品的名称、销量、售价和成本价。
4. 页面会自动计算总营业额、总销量、原料成本、总毛利和毛利率。
5. 检查下方自动生成的日报内容。
6. 点击“复制微信群日报”。
7. 打开微信，把内容粘贴到门店群或老板群。

## 本地运行方式

如果客户没有公网，也可以把项目文件夹发给对方，双击 `index.html` 打开。

开发测试时可以在项目目录运行：

```bash
python3 -m http.server 4174
```

然后打开：

http://127.0.0.1:4174/

## 适用场景

- 奶茶店每日收银汇总。
- 店员下班后给老板发日报。
- 统计爆品饮品、营业额和毛利。
- 手机端录入和查看。

## 注意事项

- 成本价按单杯原料成本填写。
- 毛利 = 营业额 - 原料成本。
- 毛利率 = 毛利 / 营业额。
- 当前版本数据不上传服务器，刷新页面后不会自动保存历史记录。
